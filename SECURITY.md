# Security Policy

## Supported Versions

Security updates are provided for the latest `master` branch state.

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue.

Report it privately through GitHub Security Advisories:

https://github.com/Fmanuel809/ai-ddd-workflow/security/advisories/new

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigations (if available)

We will acknowledge receipt within 5 business days and work with you on
validation and remediation.
