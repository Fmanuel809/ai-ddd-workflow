# Artifact Contracts

1. A0 Intake: objective, stakeholders, scope, constraints, success criteria.
2. A1 Discovery: domain signals, glossary, assumptions, open questions.
3. A2 Event Storming: event timeline, commands, actors, invariants.
4. A3 Subdomains: core/supporting/generic map with rationale.
5. A4 Contexts: bounded contexts, context map, integration contracts.
6. A5 Domain Model: aggregates, entities, value objects, policies.
7. A6 Spec/ADRs: use cases, NFRs, ADR set, acceptance criteria.
8. A7 Review: challenger findings, gaps, risks, release readiness.
